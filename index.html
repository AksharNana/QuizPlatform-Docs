Quiz Platform

This project involves creating and executing a quiz platform similar to Sporcle or Jetpunk, where users can both create and take quizzes. Additionally, users can share their quiz results with friends, and even create groups for competitions. Various types of questions should be available, such as simple question-answer, image-based, and matching questions. Quizzes should also be categorized and rated by users.

Installation

Install node_modules with npm:

  cd Quiz_Platform
  npm install

Run Locally

Go to the project directory

  cd Quiz_Platform

Start the server

  ng serve --open

Running Tests

To run tests, run the following command

  npm run test

Auto Documentation

To generate local documentation, run the following command

  npm run compodoc

To serve the genereated documentation, run the command

  npm run compodoc-serve

Continuous Integration / Continuous Deployment

Quiz Platform is using Circle CI for running tests, building and pushing to firebase hosting as well as updating the codecov page on every commit.

Deployment

The live project link is: https://quiz-platform-28fcb.web.app/

Tech Stack

Client: Angular, BulmaIO

Server: Node, Express, MongoDB
